# soy

读书笔记工具

## Usage

-   Install go

    ```bash
    export GOPATH=/opt/go
    export PATH=$GOPATH/bin:$PATH
    ```

-   npm

    ```bash
    mkdir ~/.npm-global
    npm config set prefix '~/.npm-global'
    export PATH=~/.npm-global/bin:$PATH
    ```
